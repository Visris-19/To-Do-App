## 🚀 Quick Start

<div align="center">

### 🔧 Installation Process

</div>

```bash
# 📥 1. Clone the repository
git clone https://github.com/Visris-19/To-Do-App.git
cd To-Do-App

# 📦 2. Install frontend dependencies
cd frontend
npm install

# 📦 3. Install backend dependencies
cd ../backend
npm install
```

### ⚙️ Environment Setup

<details>
<summary>📋 Click to see environment variables</summary>

**Backend (.env)**
```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_super_secret_jwt_key
SESSION_SECRET=your_session_secret_key
NODE_ENV=development
```

**Frontend (.env)**
```env
REACT_APP_API_URL=http://localhost:5000
REACT_APP_ENV=development
```

</details>

### 🎬 Launch the Application

```bash
# 🔥 Start backend server (Terminal 1)
cd backend
npm run dev

# 🚀 Start frontend (Terminal 2) 
cd frontend
npm run dev
```

## 🔧 API Documentation

<div align="center">

### 🔐 Authentication Endpoints

</div>

| Method | Endpoint | Description | Auth Required |
|--------|----------|-------------|---------------|
| `POST` | `/api/v1/auth/register` | Register new user | ❌ |
| `POST` | `/api/v1/auth/signin` | Login user | ❌ |
| `POST` | `/api/v1/auth/logout` | Logout user | ✅ |

<div align="center">

### 📋 Task Management Endpoints

</div>

| Method | Endpoint | Description | Auth Required |
|--------|----------|-------------|---------------|
| `GET` | `/api/v2/tasks` | Get all user tasks | ✅ |
| `POST` | `/api/v2/tasks` | Create new task | ✅ |
| `PUT` | `/api/v2/tasks/:id` | Update specific task | ✅ |
| `DELETE` | `/api/v2/tasks/:id` | Delete specific task | ✅ |

<div align="center">

### 👤 Profile Management Endpoints

</div>

| Method | Endpoint | Description | Auth Required |
|--------|----------|-------------|---------------|
| `GET` | `/api/v2/profile` | Get user profile | ✅ |
| `PUT` | `/api/v2/profile` | Update user profile | ✅ |
| `POST` | `/api/v2/profile/avatar` | Upload profile avatar | ✅ |

<div align="center">

### ⚙️ Settings Endpoints

</div>

| Method | Endpoint | Description | Auth Required |
|--------|----------|-------------|---------------|
| `GET` | `/api/v2/settings` | Get user settings | ✅ |
| `PUT` | `/api/v2/settings` | Update user settings | ✅ |
| `DELETE` | `/api/v2/settings/account` | Delete user account | ✅ |
